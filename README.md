# 学生成绩管理系统 python567

## 项目概述

学生成绩管理系统是一款基于Python开发，集学生、教师、管理员多角色于一体的教育信息管理工具。系统致力于简化学校成绩管理流程，提高工作效率。

## 技术栈

- **后端框架**：Flask
- **数据库**：MySQL
- **开发工具**：PyCharm
- **数据库管理工具**：Navicat

## 功能模块

- 学生档案管理
- 课程资料管理
- 选课流程管理

## 系统角色

- 学生
- 教师
- 管理员

## 运行环境

- Python
- MySQL
